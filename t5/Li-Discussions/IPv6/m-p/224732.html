Lithium doesn't use IPv6:

 

some examples:

 

telekomhilft.telekom.de is an alias for riokc95758.lithium.com.
riokc95758.lithium.com has address 46.19.168.115

 

forum.vodafone.de is an alias for vodafonede.lithium.com.
vodafonede.lithium.com has address 46.19.169.65

 

community.lithium.com is an alias for lithosphere.lithium.com.
lithosphere.lithium.com has address 208.74.207.22

 

riokc95758.lithium.com has no AAAA record

vodafonede.lithium.com has no AAAA record

lithosphere.lithium.com has no AAAA record

 

 

That is a really bad situation. More then 50% of the Vodafone and Telekom customers use already IPv6 in Germany, where the forum is used for.

 

Vodafone-Users mostly have a bad IPv4-connection (behind cgn).
